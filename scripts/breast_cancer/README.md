# Breast Cancer Scripts

Scripts for the macrophage annotation and CSDE analysis on the breast cancer dataset.

## Files

- `breast_cancer_experiment.py` — run the CSDE Poisson model on the annotated macrophages

## Workflow

See the top-level `README.md` for the full step-by-step workflow (export → annotate → run CSDE).

The annotation directories used are:

- `annotations_macrophages/` — R2 main annotations
- `annotations_macrophages_replicate` — R1 replicate annotations
